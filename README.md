## üí¨ How to Add Comments on GitHub Issues

Need help or want to share your progress? Here's how to add a comment:

1. **Navigate to the issue** - Click on the issue link above or go to the [Issues tab](https://github.com/eliastolentinont/skills-introduction-to-codeql/issues/1)
2. **Scroll to the bottom** - Find the comment box at the bottom of the issue page
3. **Write your comment** - Type your message using Markdown formatting. You can:
   - Mention users with `@username` (e.g., `@professortocat`)
   - Format text with **bold**, *italic*, or `code`
   - Add code blocks with triple backticks (\`\`\`)
4. **Preview your comment** - Click the "Preview" tab to see how it will look
5. **Submit** - Click the green "Comment" button to post

### Example Comment

```md
Hey @professortocat, I've closed and reopened an alert. What is the next step?
```
